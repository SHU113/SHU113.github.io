$$
//求平均成绩
void test03()
{
int arr[5][4] = { {56,75,78,89},{89,98,76,67},{88,99,77,66},{67,78,89,90},{98,97,96,95} };
int i = 0,j=0;
float avg[5] = { 0.0f };
int row = sizeof(arr) / sizeof(arr[0]);
int clu = sizeof(arr[0]) / sizeof(arr[0][0]);
for (i = 0; i < row; i++)
{
float sum = 0;//循环完之后要把sum初始化
for (j = 0; j < clu; j++)
{
sum += arr[i][j];
}
avg[i] = sum / clu;
cout << avg[i] << “ “;
}
}
$$

面向对象编程的三大特点：

封装：把客观事物封装成抽象的类，并且类可以把自己的数据和方法只让可信的类或者对象操作，对不可信的进行信息隐藏。类将成员变量和成员函数封装在类的内部，根据需要设置访问权限，通过成员函数，访问内部状态。

继承：所表达的是类的相关的关系，这种关系能使得对象可以继承另一对象的特征和能力。

多态：一个接口多种方法，程序在运行时才决定调用的函数。

using namespace 与局部变量发生冲突时优先选择局部变量输出
